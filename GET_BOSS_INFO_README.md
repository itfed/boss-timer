# Скрипты для получения информации о респе боссов

## Описание

Эти скрипты позволяют получать информацию о текущем состоянии таймеров респауна боссов с вашего сайта.

## Установка зависимостей

```bash
pip install requests pytz
```

## Доступные скрипты

### 1. get_boss_info.py

Подробный скрипт, который показывает полную информацию о всех боссах:

- Имя и иконка босса
- Статус (ВОЗРОЖДАЕТСЯ / ПОЯВИЛСЯ! / Не убит)
- Время до респауна
- Диапазон респауна (мин/макс)
- Время последнего убийства
- Время минимального и максимального респауна

### 2. simple_boss_info.py

Упрощенный скрипт для быстрого просмотра:

- Только номер, имя, иконка и таймер
- Удобен для быстрой проверки статуса

## Использование

### Для получения всей информации:

```bash
python get_boss_info.py
```

### Для получения упрощенной информации:

```bash
python simple_boss_info.py
```

### Настройка URL сайта

По умолчанию скрипты используют `http://localhost:5111`, но вы можете изменить это:

```python
# В начале скрипта измените переменную site_url
site_url = "https://your-site-url.onrender.com"  # Замените на ваш URL
```

### Пример использования с другим URL:

```python
# В скрипте get_boss_info.py или simple_boss_info.py
site_url = "https://your-boss-timer-site.onrender.com"
all_bosses = get_boss_respawn_info(site_url)
```

## Пример вывода simple_boss_info.py:

```

--------------------------------------------------
 1. Дух 👻 - ВОЗРОЖДАЕТСЯ: 26:32:30
 2. Петушара 🐓 - ВОЗРОЖДАЕТСЯ: 24:22:56
```

## Обработка ошибок

Скрипты содержат обработку ошибок:

- Ошибки соединения
- Ошибки парсинга JSON
- Ошибки при обращении к API

## Особенности

- Все таймеры отображаются в московском времени (UTC+3)
- Информация обновляется в реальном времени с сайта
- Поддерживает все 28 боссов из конфигурации
- Работает с текущей системой таймеров (мин/макс диапазон респауна)

## API Endpoints

Скрипты используют следующие API endpoints вашего сайта:

- `/get_boss_timers` - получение информации о всех таймерах боссов
